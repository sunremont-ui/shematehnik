# PCB Layout Module

## Purpose

Редактор топологии печатных плат. Позволяет размещать компоненты (footprints), прокладывать дорожки (traces), ставить переходные отверстия (vias), проверять DRC и экспортировать Gerber-файлы.

## Module Tree

```
PcbModule
└── (встроенные: PcbScene, FootprintLibrary, DrcPanel)
```

Зарегистрированный тип в фабрике: `PcbModule`

## Layers

| Слой | Константа | Цвет | Назначение |
|------|-----------|------|-----------|
| F.Cu | `Layer_FCu` | красный | Верхняя медь |
| B.Cu | `Layer_BCu` | синий | Нижняя медь |
| F.SilkS | `Layer_FSilkS` | жёлтый | Шёлкография |
| Edge.Cuts | `Layer_EdgeCuts` | золото | Контур платы |

Масштаб: **10 px = 1 мм** (`PCB_PX_PER_MM = 10.0`)

## Footprint Library

Встроенные шаблоны (`FootprintLibrary::instance()`):

| Имя | Описание |
|-----|---------|
| `R_0402` | Резистор 0402 SMD (2 пада) |
| `C_0805` | Конденсатор 0805 SMD (2 пада) |
| `SOT-23` | Транзистор SOT-23 (3 пада) |
| `DIP8` | DIP-8 сквозной монтаж |
| `QFP32` | QFP-32 SMD |
| `Via_0.8` | Переходное отверстие 0.8 мм |

## Usage

### Размещение компонентов

1. Выберите footprint из списка слева
2. Кликните на плате — компонент будет размещён
3. Перетащите для позиционирования
4. `R` — повернуть на 90°; `Del` — удалить

### Прокладка дорожек

1. Выберите активный слой (F.Cu / B.Cu) в выпадающем списке
2. Нажмите кнопку **Trace** или клавишу `T`
3. Кликните на пад — начало трассы
4. Кликните на второй пад — завершение
5. `Esc` — отменить текущую трассу

Дорожки привязываются к сетке **0.25 мм**.

### Переходные отверстия

- Нажмите `V` или кнопку **Via** для добавления vias
- Параметры по умолчанию: внешний диаметр 1.2 мм, отверстие 0.6 мм

### Undo / Redo

`Ctrl+Z` / `Ctrl+Y` — поддерживается для трасс, vias и размещения footprints.

## DRC (Design Rule Check)

Нажмите **Run DRC** для проверки:

- Перекрытия медных элементов (clearance violations)
- Неподключённые цепи (airwires)
- Footprints без привязки к netlist

Результаты выводятся в DRC-панели снизу.

## Netlist Import

PCB-модуль подписывается на EventBus-событие `netlist.ready`. При экспорте нетлиста из SchematicModule → NetlistModule, PCB автоматически получает соединения и отображает airwires между несоединёнными падами.

## Gerber Export

Кнопка **Export Gerber** открывает диалог выбора папки. Генерируются файлы:

```
project_F_Cu.gbr
project_B_Cu.gbr
project_F_SilkS.gbr
project_Edge_Cuts.gbr
project_drill.drl
```
