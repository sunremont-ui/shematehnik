# Overview

**Universal Controller Platform (UCP)** — открытое десктопное приложение для сквозного проектирования электронных устройств. Замена связке из 4–5 разрозненных программ одним модульным инструментом.

## Проблема

Сегодня для создания устройства нужно:
1. **KiCad** — принципиальная схема и PCB
2. **FreeCAD** — 3D-модель корпуса
3. **SquareLine Studio** — UI экранки (LVGL)
4. **Proteus / Wokwi** — симуляция МК
5. **Draw.io / Mermaid** — описание протоколов

Каждая программа — свой формат, свои хоткеи, своя логика. Перенос данных между ними — ручной экспорт/импорт.

## Решение

UCP — один инструмент, где **каждая задача — это модуль**, а модули могут быть вложены друг в друга:

```
Schematic (модуль)
  ├── SymbolEditor (подмодуль)
  ├── WireTool (подмодуль)
  ├── NetlistGenerator (подмодуль)
  └── SPICESimulator (подмодуль)
```

Проект = файл `.ucp` = JSON-дерево всех модулей и их состояния.

## Целевая аудитория

- Инженеры-электронщики, делающие устройства «от идеи до корпуса»
- Студенты и радиолюбители
- Малые производства (теплицы, станки, роботы)

## Статус

Pre-alpha. Работает ядро (дерево модулей, фабрика, event bus). Написан модуль-пример SchematicEditor с 4 подмодулями.
